# DCE 5.0 v0.38.0-20260131 Enterprise Released

This page explains the latest features, improvements and bug fixes, included
in DCE 5.0 v0.38.0-20260131 Enterprise. DCE 5.0 modularizes each product module,
and the versions included in this release are the default versions
in the DCE 5.0 installer. If you need to independently upgrade or replace each module, contact
Daocloud's support team to download the proper versions and upgrade. Also refer to
[DCE 5.0 Enterprise Deployment Requirements](../../install/commercial/deploy-requirements.md).

## Upgrade Notes

Starting from Microservice Engine v0.52.1, the AI Gateway Higress has been upgraded to v2.1.9,
which includes changes to its CRDs. Before upgrading, please manually update the CRDs in the target cluster.

1. Download the `skoala-init` Chart package  
2. Locate the corresponding YAML file according to the Chart package path  
3. Run the following command to update the CRDs

```sh
kubectl apply -f skoala-init/charts/contour-provisioner/crds/higress.gen.yaml
```

Refer to the [Microservice Engine Upgrade Notes](../../skoala/intro/upgrade-notes.md).
